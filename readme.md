1.打开软件 提示需要激活为正常状况
2.打开我发给你的windows一键激活（不要修改东西）
3.执行那个双击这里激活一声 等他自己执行完 他会自动删除文件夹 删除成功以后
4.关闭软件 重新打开 （没有提示激活就算没问题了）
5.打开软件打开一个项目 点击help 点击register 看看是否显示2099年到期
